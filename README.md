# Women-Loan-Scheme

**Women-Loan-Scheme** is a java-based application designed to manage loan processes efficiently. This project focuses on **Multithreading** and ** Thread Safety** to handle multiple loan requests concurrently.

---
## Key Technical Features

* **Multithreading :** Used the "Thread" class to process multiple loan applications at the same time .
* **Synchronization :** Implemented **two sychronized method** to ensure data integrity and prevent race conditions when updating account balances .
* **Sleep Method :** Utilizes "Thread.sleep()"" to simulate real-world processing delays, making the loan approval flow more realistic .
* **Execution :** Utilizes the ".start()" method to initiate thread exeution and manage concurrent tasks .
